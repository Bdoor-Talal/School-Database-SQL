# School-Database-SQL
SQL database project for managing students, subjects, and grades

## Project Description
This project demonstrates how to create a SQL database for students, subjects, and grades. It includes:
- **Students** table with information like name, age, level, GPA, gender, and track.
- **Subjects** table with subject name, level, and track.
- **Grades** table linking students to subjects and storing their grades.
- SQL queries to retrieve important data such as top students, averages, and student counts per level.

##  Technologies Used
- SQL Server
- SQL queries for creating tables, inserting data, and querying information.

##  Main Tables
1. **Students**: Contains student information.
2. **Subjects**: Contains subjects for each track and level.
3. **Grades**: Contains grades of students in various subjects.
